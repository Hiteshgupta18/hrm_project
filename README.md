# HRM Fuel It Online

Human Resource Management System

## Project Structure

- `/frontend` - React application
- `/backend` - Express.js API server

## Setup

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
npm install
npm start
```

## Features

- Employee Management (Add, View, Edit)
- User Authentication
- Search and Filter
- Pagination
